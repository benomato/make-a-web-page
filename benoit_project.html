Lesson 1: The Basics of the web and HTML

Some context

World Wide Web: a collection of HTML documents.

HTML: hypertext Markup Language.

Hyperlinks: what makes the web web like.

What type of files can be found on the web?

  • Plain text
  • HTML
  • Images
  • Videos
  • Music

Major pieces

The browser makes requests via internet to servers. These requests are using a protocol called HTTP and the servers respond with files that the browser displays.

HTML

HTML is made of:

  • text content: What I see
  • markup: What it looks like / how it is arranged
  • references to other documents: e.g: images, videos.
  • links to other pages

Plain text is plain text in HTML.

HTML Markups

HTML is made of things called tags.

We can write it like this: <name>contents</name>

list of the principal tags:

  • <b> -----> bold
  • <em> -----> emphasis, italics
  • <a href="....."> -----> anchor, anchors are made for links
  • <img src="url" alt="text"> - images
  • <br> -----> break, inline tag, go to next line
  • <p> -----> paragraph, block tag, makes an invisible box
  • <span> -----> normal tag, inline
  • <div> -----> normal tag, block
  • <! DOCTYPE HTML> -----> Doctype to use at the beginning of every html file.
  • <html> -----> opening html tag
  • <head> -----> metadata and random stuff
  • <title> -----> Title that of the page, see it in tab
  • <body> -----> The actual content of the HTML document

Lesson 2: Creating a structured document

A website is like a house:

  • HTML files are like the structure.
  • CSS files are then the style of your house.
  • JS files are the interactive components(e.g: television remote).
  • It also exists some rules to follow(e.g: DOM).
  • Eventually, some tools can help us build our websites(e.g: Chrome developer tools).

The DOM(Document object model) is the tree like structure of an html page that a browser interprets.

Everything in a webpage is a box.
Boxify a design means break down it in several boxes.
To help create this box in the html, we can use the div tag.
To bring style to this "div", we add the attribute class that allows to use CSS.

Lesson 3: Adding CSS Style to HTML structure

Welcome to CSS

CSS is a language that avoids programmers to repeat things over and over. Using html tags and other selectors, you can quickly add the same style to different section of your HTML code. It saves a lot of time and lines of code.

In the CSS acronym, C stands for cascading. It is a really interesting rules that means that the most specific CSS rule applies to an HTML element.

To style a page using CSS, you need to choose a selector apply some CSS properties. For example, if you want to change the color of an h2 to red you do:

.h2  {
color: red;
}

Everything in a webpage is a box

A web page could be described as a box model: it is a box, that has a box that has a box etc..

The box model include, from outside in: margin, border, padding, content. We can find a great explanation here: W3schools

The "True" width of an HTML element is the sum of content + padding + border. The box-sizing CSS property specifies that elements should have padding and border included in the element's total width and height. It doesn't include margins

Display blocks in the same line

CSS property display sets to value flex allows to display block elements on the same line. For some HTML block elements in a container, CSS property display must be applied to the container so the block elements could be on the same page.

CODE, TEST AND REFINE

One you received the deign mockup, you can follow these steps:

  1. Look for the natural boxes and try to boxify the design
  2. Look for styles that repeat
  3. Write the HTML
  4. Apply styles (from the biggest to the smallest)
  5. Test and fix
  6. Repeat steps 3 to 5 until your webpage matches the design
